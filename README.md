# Letmeask

![letmeask](https://user-images.githubusercontent.com/54786785/124206657-24235680-daba-11eb-977e-02b80d7873f7.png)


Projeto da semana NLW 06, criação de salas de perguntas e respostas em tempo real.

## Tecnologias
Esse projeto foi desenvolvido com as seguintes tecnologias:
- [React](https://pt-br.reactjs.org/)
- [Typescript](https://www.typescriptlang.org/)
- [Firebase com banco de dados](https://firebase.google.com/)
